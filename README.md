# Movie Review Sentiment Analysis 🎬🤖

This project is a Machine Learning model that classifies movie reviews as Positive or Negative using NLP techniques.

## 🔧 Tech Stack
- Python
- pandas
- scikit-learn
- nltk
- Streamlit

## 🚀 Features
- Text preprocessing
- TF-IDF vectorization
- Logistic Regression model
- Streamlit web app

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
